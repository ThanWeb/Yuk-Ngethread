FUNGSI MELIBATKAN REST API
Base URL https://forum-api.dicoding.dev/v1/#/

------------------------------

getAccessToken(): mendapatkan access token dari local storage.

putAccessToken(): menyimpan access token dari local storage.

register({ name, email, password (minimun 6 karakter) }): registrasi pengguna baru.
method POST, endpoint /register

login({ email, password }): proses autentikasi untuk mendapatkan access token.
method POST, endpoint /login

getAllUsers(): mendapatkan seluruh daftar pengguna.
method GET, endpoint /users

getOwnProfile(): mendapatkan pengguna yang terautentikasi.
method GET, endpoint /users/me

createThread({title, body, category}): membuat thread baru. 
method POST, endpoint /threads

getAllThreads(): mendapatkan seluruh daftar thread. 
method GET, endpoint /threads

getDetailThread(id): mendapatkan detail dari sebuah thread. 
method GET, endpoint /threads/<threadId>

createCommentThread(content, id) : menambahkan komentar pada sebuah thread.
method POST, endpoint /threads/<threadId>/comments

giveUpVoteThread() : menambahkan voting naik pada sebuah thread.
method POST, endpoint /threads/<threadId>/up-vote

giveDownVoteThread : menambahkan voting turun pada sebuah thread.
method POST, endpoint /threads/<threadId>/down-vote

giveNeutralVoteThread : menambahkan voting netral pada sebuah thread.
method POST, endpoint /threads/<threadId>/neutral-vote

giveUpVoteComment() : menambahkan voting naik pada sebuah komentar.
method POST, endpoint /threads/<threadId>/comments/<commentId>/up-vote

giveDownVoteComment : menambahkan voting turun pada sebuah komentar.
method POST, endpoint /threads/<threadId>/comments/<commentId>/down-vote

giveNeutralVoteComment : menambahkan voting netral pada sebuah komentar.
method POST, endpoint /threads/<threadId>/comments/<commentId>/neutral-vote

getLeaderboards : menampilkan daftar pengguna dengan skor tertinggi.
method GET, endpoint /leaderboards
